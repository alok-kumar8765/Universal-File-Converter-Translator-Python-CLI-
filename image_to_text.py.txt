Using Download and install Tesseract
	> https://digi.bib.uni-mannheim.de/tesseract/
		>>Tesseract provides a handy command-line tool that you can use to perform OCR on images. 
		  After you install Tesseract, open a CLI window, navigate to the folder where the image 
		  file whose text you want to extract is located, and run the following command:  

			>>> tesseract  out

		  This command extracts text from the specified image and saves the text in the out.txt file. 
		  To use Tesseract with Python, proceed to the next step to install the required Python packages.

Extract Text From Images in Python Using EasyOCR

Requirements.txt -->
	- pip install torch torchvision torchaudio
	- pip install easyocr

Code -->
	import easyocr
	reader = easyocr.Reader(['en'])
	result = reader.readtext('test.jpg', detail = 0)
	print(result)

Using OCRopus -->
	import ocrolib
	image = ocrolib.read_image('image.jpg')
	text = ocrolib.ocr(image)
	print(text)

Using PyOCR -->
	import PIL.Image 
	import pyocr
	import pyocr.builders
	tools = pyocr.get_available_tools() if len(tools) == 0: print("No OCR tool found.") exit(1)                                    ocr_tool = tools[0]
	image = PIL.Image.open('image.jpg')
	text = ocr_tool.image_to_string( image, builder=pyocr.builders.TextBuilder() )
	print(text)
